# react_demo
基于react的前端框架，主要包含react+redux+antd

## Install
    npm install

## Run
	npm run dev  （一般开发模式，打开http://localhost:3000/）
	npm run dist  (发布时打包，在page/main/dist 文件夹下生成打包文件) 
	npm run hot  （热替换开发模式，打开http://localhost:8888/）
